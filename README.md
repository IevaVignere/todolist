# todolist

Vienkārša lietotne, kas radīta skolēniem un studentiem ikdienas uzdevumu, mājasdarbu un projektu uzskaitīšanai. Izstrādāta ar Python un CustomTkinter, nodrošinot modernu grafisko saskarni.

## Galvenās Funkcijas
Pievienot uzdevumus: ātri pievieno jaunus uzdevumus, iebūvētā validācija novērš tukšu uzdevumu izveidi.
Atzīmēt kā izpildītu: ar pogu "pabeigts" atzīmē pabeigtos darbus (teksts tiek pārsvītrots).
Dzēst: ar "dzēst" pogu noņem uzdevumus, kad tie vairs nav aktuāli.
Datu saglabāšana: visi uzdevumi tiek automātiski saglabāti lokālā `tasks.json` failā, nodrošinot, ka dati nepazūd pēc programmas aizvēršanas.
Tumšais/Gaišais režīms: programma automātiski pielāgojas operētājsistēmas dizainam.
